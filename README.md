A simple web-based dice roller built using HTML, CSS, and JavaScript.  
This project generates random dice values and displays the corresponding dice images.

## Features
- Roll one or multiple dice
- Random number generation
- Dynamic dice image display
- Clean and simple user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the repository.
2. Open the `index.html` file in your browser.
3. Enter the number of dice.
4. Click **Roll Dice** to see the results.
